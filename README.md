
# תפזורת שמות – משחק רשת (Hebrew Word Search)

משחק תפזורת שמות מותאם לנייד. ניתן לארח כעמוד סטטי ב‑GitHub Pages ולקבל קישור לשיתוף.

## מה יש כאן
- `index.html` – עמוד המשחק
- `style.css` – עיצוב רספונסיבי מותאם לנייד
- `script.js` – לוגיקת המשחק: יצירת מטריצה, בחירת מילים, בדיקת תשובות, משחק חדש

## איך מעלים ל‑GitHub ומקבלים קישור
1. צרו מאגר (Repository) חדש ב‑GitHub, למשל: `hebrew-wordsearch`.
2. העלו לתוכו את שלושת הקבצים (`index.html`, `style.css`, `script.js`).
3. עברו ל‑**Settings → Pages**.
4. תחת **Build and deployment**, בחרו:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` / `master`, תיקייה `/ (root)`
5. לחצו **Save**. לאחר דקה‑שתיים, יופיע קישור בסעיף **GitHub Pages**. זהו הקישור לשיתוף ושימוש מהנייד.

> טיפ: אם המאגר לא ציבורי, GitHub Pages לא יפעל. ודאו שהמאגר **Public** או השתמשו ב‑Netlify/Vercel.

## איך משחקים
- גררו באצבע (בסלולרי) או בעכבר (במחשב) מראשית המילה עד סופה — בכל 8 הכיוונים, כולל הפוכים.
- תקבלו סימון ירוק אם פגעתם במילה. לא נכון? התא יתנדנד קלות באדום.
- לחצו על **"משחק חדש"** כדי לערבב ולהתחיל משחק רענן.
- **"איפוס סימונים"** להסרת כל ההצלבות (ללא שינוי הלוח).

## שינוי רשימת השמות
רשימת השמות מופיעה ב‑`index.html` בתוך `window.HEBREW_NAMES` ב‑`<script>`. ניתן לערוך/להחליף שם.

## הערות לשפה העברית
- האותיות בלוח מוצגות בצורתן הרגילה (לא‑סופית), למעט תאים שהם **סוף מילה** ואינם משותפים למילה אחרת — שם תוצג הצורה הסופית המתאימה (ך/ם/ן/ף/ץ).
- אין רווחים בין שם פרטי לשם משפחה, ומקפים הוסרו לצורך המשחק.
